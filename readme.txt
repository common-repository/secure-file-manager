== Description ==
Highly secure, powerful and most beautiful WordPress File Manager plugin with multiple beautifully designed themes.

Access your server files in the most secure way with the best WordPress File Manager plugin. Multiple beautiful themes to fit your need. Remove dependency on FTP or SFTP application. Saves time and effort. Very easy to use. Manage access to files with User ID or User Role.

Secure File Manager makes managing your server files easier than ever.

**Create, Delete, Edit, Download, Upload, Compress (Zip):** All the file management basic tasks you need to do on a daily basis.

**FTP and SFTP Support:** Connect to external servers through FTP or SFTP.

**Manage Everything from one place:** Manage all your server files from one place - your WordPress dashboard.

**Access Management:** Manage and set access to server files by User Role or User ID.

You can define the **ONLY** user who can access the File Manager for higher security.

== Installation ==
1. Download the plugin zip file.
2. Extract the downloaded zip file.
3. Upload the \"wp-secure-file-manager\" directory to the \"/wp-content/plugins/\" directory.
4. Activate the plugin through the \"Plugins\" menu in WordPress.
